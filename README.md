2.  **Check the Documentation:** Many common questions are answered in our Help Center/Documentation.

## How to Report a Bug

If you find a bug, please create a **[New Issue](https://github.com/fuliginheart/issue-tracking/issues/new)** and include the following details:

* **Description:** A clear and concise description of what the bug is.
* **Steps to Reproduce:**
    1. Go to '...'
    2. Click on '...'
    3. Scroll down to '...'
    4. See error
* **Expected Behavior:** What you expected to happen.
* **Actual Behavior:** What actually happened.
* **Screenshots/Video:** (Optional but highly recommended)
* **Environment:**
    * OS: [e.g. Windows 10, macOS Big Sur]
    * Browser: [e.g. Chrome, Safari]
    * Version: [e.g. 2.1.4]

## How to Request a Feature

I love hearing ideas for how to improve my stuff! When suggesting a feature, please include:

* **The Problem:** What pain point are you trying to solve?
* **The Solution:** How would you like the feature to work?
* **Alternatives:** Have you considered any other way to solve this?
